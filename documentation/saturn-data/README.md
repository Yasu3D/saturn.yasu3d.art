# SATURN DATA
SATURN DATA is the core library for SATURN and SATURN EDIT.
It defines all chart notation and custom content types, 
and handles their serialization and deserialization.

> This documentation page serves as a general guide and surface level explanation of SATURN DATA.  
Nearly everything has XML documentation in the code, which will usually be more accurate and useful for developers than this page.

To learn more about how to use SATURN DATA, please refer to one of the available guides.
