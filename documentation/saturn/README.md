# SATURN
SATURN is a circular touch slider rhythm game that has a really cool ring, developed in Unity 6.

## Download
Downloads are currently unavailable.

## Recommended System Requirements
@1080p60hz
- OS: Windows 11, Windows 10 21H1 or newer, Linux
- CPU: **Intel Core i5-6500** equivalent or better.
- GPU: **NVIDIA GeForce GTX 1050Ti** equivalent or better.
- RAM: **4GB** or more.
- SSD: **1GB** or more.

> [!NOTE]
> SATURN is not very demanding on the GPU and _may_ even run on integrated graphics.  

> [!IMPORTANT]
> Storage and Memory usage both depend on the amount of content that has been added.  

> [!IMPORTANT]
> Memory usage will increase over time as content is loaded and cached.
