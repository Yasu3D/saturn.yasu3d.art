# Shortcut Notation Conventions

### Keyboard
| Notation                                                                  | Explanation                                                    |
|---------------------------------------------------------------------------|----------------------------------------------------------------|
| <kbd>G</kbd>                                                              | The "G" key by itself (as if you were typing a lowercase "g"). |
| <kbd>Shift</kbd>, <kbd>Ctrl</kbd>, <kbd>Alt</kbd>                         | Modifier keys.                                                 |
| <kbd>Ctrl</kbd>-<kbd>C</kbd>, <kbd>Ctrl</kbd>-<kbd>Alt</kbd>-<kbd>S</kbd> | Keys pressed simultaneously.                                   |
| <kbd>0</kbd> to <kbd>9</kbd>                                              | The number keys on the row above the letters.                  |
| <kbd>Numpad0</kbd> to <kbd>Numpad9</kbd>, <kbd>NumpadPlus</kbd>           | The keys on the separate numeric keypad.                       |

### Mouse
| Notation                                 | Explanation                                            |
|------------------------------------------|--------------------------------------------------------|
| <kbd>LMB</kbd>                           | Left Mouse Button                                      |
| <kbd>RMB</kbd>                           | Right Mouse Button                                     |
| <kbd>MMB</kbd>                           | Middle Mouse Button (Usually clicking the mouse wheel) |
| <kbd>WheelUp</kbd>, <kbd>WheelDown</kbd> | Scrolling the mouse wheel.                             |

> [!NOTE]
> These shortcut notation conventions are inspired by the Blender 5.0 Manual.
