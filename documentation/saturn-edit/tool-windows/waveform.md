# Waveform
The Waveform Tool Window allows you to see a waveform of the currently loaded audio.

<img src="media/waveform.png">

## Color Coding
The judgement line is represented by a <span style="color: #ff0000;">red</span> line.  

Measure Line Notes are represented by a <span style="color: #FFFFFF;">white</span> line.  

Beat Line Notes are represented by a <span style="color: #AAAAAA;">grey</span> line.
