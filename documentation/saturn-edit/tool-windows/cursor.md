# Cursor
The Cursor Tool Window lets you change the current Timestamp and Cursor Note shape.

<img src="media/cursor.png">

## Measure
The current Measure the Cursor is on.

## Beat
The current fraction of the Measure the Cursor is on.  
This value is defined by two numbers, the Beat Number and Beat Division.

### Beat Number
The number of steps into a Measure the Timestamp is.

### Beat Division
The number of steps a Measure is divided into.

## Position
The Position of the Cursor.

## Size
The Size of the Cursor.
