# Adding Content

Saturn allows users to import various types of custom content. All user-imported data is located in the `StreamingAssets` folder:

```file tree
.
└── Saturn
    └── Saturn_Data
        └── StreamingAssets
            ├── Data
            │   ├── ConsoleColor
            │   ├── Emblem
            │   ├── Icon
            │   ├── Navigator
            │   ├── NoteSound
            │   ├── Plate
            │   ├── SystemSound
            │   └── Title
            ├── Led
            ├── Locales
            ├── StageUp
            └── Music
```
